# FIAP BANK - ATM

## 📌 Descrição

Este projeto simula um terminal de autoatendimento bancário (ATM) desenvolvido em Java, executado via console.

A aplicação permite que o usuário cadastre uma senha e utilize funcionalidades básicas de um caixa eletrônico, como consulta de saldo, depósito, saque e alteração de senha.

---

## ⚙️ Funcionalidades

* Cadastro de senha (4 dígitos)
* Login com validação de tentativas
* Consulta de saldo
* Depósito
* Saque (com validação de saldo)
* Alteração de senha
* Tratamento de erros básicos de digitação

---

## ▶️ Como executar

1. Abra o projeto em uma IDE (IntelliJ ou VS Code)
2. Compile o arquivo `FiapBankAtm.java`
3. Execute o programa
4. Siga as instruções exibidas no terminal

---

## 🧠 Tecnologias utilizadas

* Java
* Scanner (entrada de dados)
* Estruturas de controle (`if`, `while`, `switch`)
